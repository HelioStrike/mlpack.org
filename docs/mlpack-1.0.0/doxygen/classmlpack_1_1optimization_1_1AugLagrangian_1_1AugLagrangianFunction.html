class mlpack::optimization::AugLagrangian< LagrangianFunction >::AugLagrangianFunction

This is a utility class, which we will pass to L-BFGS during the optimization.

We use a utility class so that we do not have to expose Evaluate() and Gradient() to the AugLagrangian public interface; instead, with a private class, these methods are correctly protected (since they should not be being used anywhere else).
