126  // Column i of 'diffs' is the difference between x.col(i) and the mean.

128 
129  // Now, we only want to calculate the diagonal elements of (diffs' * cov^-1 *
130  // diffs). We just don't need any of the other elements. We can calculate
131  // the right hand part of the equation (instead of the left side) so that
132  // later we are referencing columns, not rows -- that is faster.
