mlpack_cli_main.hpp
Author
Ryan Curtin

This file, based on the value of the macro BINDING_TYPE, will define the macro MLPACK_CLI_MAIN() accordingly. If BINDING_TYPE is not set, then the behavior will be that MLPACK_CLI_MAIN() is equivalent to int main(int argc, char** argv) with CLI::ParseCommandLine() automatically called on entry and CLI::Destroy() automatically called on exit.

This file should only be included by a program that is meant to be a command-line program or a binding to another language.
